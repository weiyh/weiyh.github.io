Climate Change News Line Bot

氣候變遷即時新聞LINE機器人

Web app
  • Completed on: October 2019
  • Skills / Tools Used: Java, Spring Framework (Spring Boot, Spring Scheduler), LINE Messaging API

A LINE Bot which retrieves Google Alerts' RSS feeds periodically (every hour) and sends feeds information to LINE user. This Line Bot is designed for reading news related to climate change.

定時將Google快訊(Google Alerts) RSS訊息傳至Line聊天室，用於追蹤氣候變遷相關的新聞。
